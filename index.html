How it works

Up and running in minutes

VibOps deploys as a lightweight control plane next to your existing infrastructure. No agents required on GPU nodes. No data exfiltration.

01

Deploy VibOps

Self-hosted via Helm or Docker Compose. Deploys inside your infrastructure — on-prem, cloud, or hybrid.

02

Connect your clusters

Install a lightweight Connect Gateway on each cluster. Auto-discovers namespaces, deployments, and GPU resources via kubectl.

03

Operate in natural language

Use the VibOps console or connect Claude Desktop / Cursor via pip install vibops-mcp. Your AI assistant now speaks natively to your stack.

04

Full audit trail

Every operation is logged with user, timestamp, and exact command executed. Immutable. Compliance-ready from day one.
